# Feature Selection UI

A simple and responsive feature selection interface built using HTML and CSS. Each feature is presented as a card with a custom-styled checkbox, allowing users to select multiple options in an interactive and visually appealing layout.

## 🚀 Features

- Responsive grid layout
- Interactive feature cards
- Custom styled checkboxes
- Hover animations and effects
- Clean and minimal UI design
- Fully built using HTML and CSS (no JavaScript)


## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Transitions)

## 📂 Project Structure


project-folder/
│── index.html
│── styles.css

## 🎯 Purpose

This project was built as a practice exercise to improve understanding of:
- CSS Grid layout
- Custom form element styling
- Hover and transition effects
- UI design fundamentals
